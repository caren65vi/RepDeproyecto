# Proyecto Final

## Requisitos

Este proyecto usa Node.js `24.14.1`.

Para instalar la misma version con `nvm`:

```bash
nvm install
nvm use
```

Si usas Windows, puedes instalar `nvm-windows` y ejecutar:

```bash
nvm install 24.14.1
nvm use 24.14.1
```

Luego instala las dependencias:

```bash
npm install
```

Si tienes otra version de Node.js, `npm install` se detendra porque el proyecto tiene `engine-strict=true`.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```
